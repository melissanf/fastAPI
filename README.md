# FastAPI BMI Calculator

A simple REST API built with **FastAPI** that calculates BMI (Body Mass Index) based on weight and height inputs.
Includes a connected HTML frontend for easy interaction.

## Features

- BMI calculation via query parameters
- Input validation with FastAPI's `Query`
- Category classification (Underweight / Normal / Overweight / Obese)
- Simple HTML frontend connected to the API

## Stack

- Python · FastAPI · Uvicorn · HTML/CSS/JS
